useStateは、まず現在のリストのコピーを作成→今回の処理の内容をコピーしたリストに反映→set関数を呼び出して状態を最新のリストの内容で更新（更新時はコンポーネントが再レンダされる）。これは覚えた。配列のコピーはjsとtsで記載が異なる！
``` JavaScript
// JavaScript

```
``` TypeScript
// TypeScript
const copylist = [...originallist];
```

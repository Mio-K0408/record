
# 概要
スプシで管理していた学習記録をitHubでも管理する
時間の管理は関数が使えるスプシを結局使う必要があるので二重管理になるが、日々の学習記録をもう少し詳細に残したいのでお試し。

## フォルダ構成
細かいとはこれから決めるがいったん以下の通り
- 2024
    - 10月
        - 1週目
        - 2週目
        ・・・
    - 11月
        - 1週目
- 2025
    - 1月
    ・・・
    
## ルール
いったんは自分の負担にならない程度に緩くやる
